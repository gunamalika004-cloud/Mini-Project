# City Traffic Navigation System Using Graph Data Structure

## Project Overview

The City Traffic Navigation System is a Python-based mini project that helps users find the shortest route between two locations in a city. The system represents the city road network using a Graph Data Structure, where intersections are represented as vertices (nodes) and roads are represented as edges. Dijkstra's Algorithm is used to calculate the shortest path efficiently.

---

## Problem Statement

Traffic congestion and inefficient route planning can increase travel time and fuel consumption. Drivers often need a reliable system to identify the shortest route between locations. This project addresses this problem by implementing a graph-based navigation system.

---

* Represent city roads using a Graph Data Structure.

## Methodology

1. Create a graph representing the city road network.
2. Represent intersections as nodes.
3. Represent roads as edges with distance weights.
4. Accept source and destination locations.
5. Apply Dijkstra's Algorithm.
6. Calculate the shortest path.
7. Display the optimal route and total distance.
